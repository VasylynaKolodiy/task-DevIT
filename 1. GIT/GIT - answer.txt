Щоб надіслати свій робочий код до репозиторію git@example.com:example/test.git, потрібно виконати такі кроки:

1) Відкрити термінал і викорисовуючи команду cd (change directory) перейти у папку, де знаходиться робочий код.

2) Ініціалізувати Git-репозиторій за допомогою команди: 
git init

3) Додати та зафіксувати зміни:
git add .
git commit -m "Initial commit"

4) Зв'язати локальний репозиторій з віддаленим репозиторієм:
git remote add origin git@example.com:example/test.git

5) Відправити код у гілку master віддаленого репозиторію:
git push -u origin master

Опція -u встановлює відповідність між моєю локальною гілкою master та гілкою master на сервері.
Це дозволить потім використовувати git pull та git push без вказівки гілки.
